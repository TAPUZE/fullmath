פונקציה רציונלית (חדו"א) рационал

לומדים: חקירת פונקציה רציונלית

פונקציה רציונלית היא פונקציה מהצורה \(f(x) = \frac{P(x)}{Q(x)}\), כאשר \(P(x)\) ו-\(Q(x)\) הם פולינומים, ו-\(Q(x) \neq 0\).

1. תחום הגדרה

תחום ההגדרה של פונקציה רציונלית הוא כל המספרים הממשיים, פרט לערכי \(x\) שמאפסים את המכנה (\(Q(x)=0\)).

2. אסימפטוטות

אסימפטוטות הן ישרים שהגרף של הפונקציה מתקרב אליהם ככל ש-\(x\) (או \(y\)) שואף לאינסוף או לערך מסוים.

  • אסימפטוטה אנכית: תיתכן בערכי \(x\) המאפסים את המכנה אך אינם מאפסים את המונה. משוואת האסימפטוטה האנכית היא \(x=k\), כאשר \(k\) הוא שורש של המכנה שאינו שורש של המונה.
  • אסימפטוטה אופקית: נקבעת על ידי השוואת החזקה הגבוהה ביותר במונה (\(n\)) לחזקה הגבוהה ביותר במכנה (\(m\)).
    • אם \(n < m\): האסימפטוטה האופקית היא \(y=0\) (ציר ה-x).
    • אם \(n = m\): האסימפטוטה האופקית היא \(y = \frac{a_n}{b_m}\) (יחס המקדמים של החזקות הגבוהות).
    • אם \(n > m\): אין אסימפטוטה אופקית (עשויה להיות אסימפטוטה משופעת, אך זה מעבר לחומר הלימוד ברמה זו).

דוגמה (ממסמך המחקר): מצא את האסימפטוטות של הפונקציה \(f(x) = \frac{2x+1}{x-3}\).

פתרון:

אסימפטוטה אנכית: המכנה מתאפס כאשר \(x-3=0 \Rightarrow x=3\).
המונה ב-\(x=3\) הוא \(2(3)+1 = 7 \neq 0\).
לכן, \(x=3\) היא אסימפטוטה אנכית.

אסימפטוטה אופקית: החזקה הגבוהה ביותר במונה היא \(x^1\) (המקדם הוא 2). החזקה הגבוהה ביותר במכנה היא \(x^1\) (המקדם הוא 1).
מכיוון שהחזקות שוות (\(n=m=1\)), האסימפטוטה האופקית היא יחס המקדמים: \(y = \frac{2}{1} = 2\).

3. נגזרת של פונקציה רציונלית (כלל המנה)

כדי לגזור פונקציה רציונלית \(f(x) = \frac{u(x)}{v(x)}\), משתמשים בכלל המנה:

f'(x) = \left(\frac{u}{v}\right)' = \frac{u'v - uv'}{v^2}

כאשר \(u'\) היא הנגזרת של המונה ו-\(v'\) היא הנגזרת של המכנה.

דוגמה (ממסמך המחקר): גזור את הפונקציה \(f(x) = \frac{x+2}{x-1}\).

פתרון:

נסמן: \(u(x) = x+2 \Rightarrow u'(x) = 1\).

נסמן: \(v(x) = x-1 \Rightarrow v'(x) = 1\).

נשתמש בכלל המנה:

f'(x) = \frac{u'v - uv'}{v^2} = \frac{1 \cdot (x-1) - (x+2) \cdot 1}{(x-1)^2}
f'(x) = \frac{x-1 - x-2}{(x-1)^2} = \frac{-3}{(x-1)^2}

4. חקירת פונקציה רציונלית

החקירה דומה לחקירת פונקציית פולינום, אך כוללת גם התייחסות לתחום ההגדרה ולאסימפטוטות:

  • תחום הגדרה.
  • נקודות חיתוך עם הצירים.
  • אסימפטוטות (אנכיות ואופקיות).
  • נקודות קיצון: מצא נקודות חשודות (\(f'(x)=0\)) וסווג אותן (בעזרת טבלה או נגזרת שנייה, תוך התחשבות בתחום ההגדרה).
  • תחומי עלייה וירידה.
  • (לעיתים נדרש גם שרטוט סקיצה של הגרף).

דוגמה (ממסמך המחקר): חקור את הפונקציה \(f(x) = x + \frac{1}{x}\) (עבור \(x \neq 0\)) ומצא נקודות קיצון ותחומי עלייה וירידה.

פתרון:

1. תחום הגדרה: \(x \neq 0\).

2. נגזרת: \(f(x) = x + x^{-1} \Rightarrow f'(x) = 1 - 1x^{-2} = 1 - \frac{1}{x^2} = \frac{x^2-1}{x^2}\).

3. נקודות חשודות לקיצון (\(f'(x)=0\)): \(\frac{x^2-1}{x^2} = 0 \Rightarrow x^2-1 = 0 \Rightarrow x^2 = 1 \Rightarrow x = \pm 1\).

4. נגזרת שנייה: \(f'(x) = 1 - x^{-2} \Rightarrow f''(x) = -(-2)x^{-3} = \frac{2}{x^3}\).

  • עבור \(x=1\): \(f''(1) = \frac{2}{1^3} = 2 > 0\) (מינימום).
    \(f(1) = 1 + \frac{1}{1} = 2\). נקודת מינימום: \((1,2)\).
  • עבור \(x=-1\): \(f''(-1) = \frac{2}{(-1)^3} = -2 < 0\) (מקסימום).
    \(f(-1) = -1 + \frac{1}{-1} = -1 - 1 = -2\). נקודת מקסימום: \((-1,-2)\).

5. תחומי עלייה וירידה (לפי סימן \(f'(x) = \frac{x^2-1}{x^2}\). המכנה \(x^2\) תמיד חיובי בתחום ההגדרה, לכן הסימן נקבע ע"י המונה \(x^2-1\)):

\(x^2-1 > 0 \Rightarrow x^2 > 1 \Rightarrow x > 1\) או \(x < -1\) (עלייה).

\(x^2-1 < 0 \Rightarrow -1 < x < 1\) (וגם \(x \neq 0\)) (ירידה).

תשובה: מקסימום \((-1,-2)\), מינימום \((1,2)\).
תחומי עלייה: \(x < -1\) או \(x > 1\).
תחומי ירידה: \(-1 < x < 0\) או \(0 < x < 1\).

מתרגלים ✍️

תרגיל 1: מצא את תחום ההגדרה והאסימפטוטות (אם יש) של הפונקציה \(f(x) = \frac{3x}{x^2-4}\).

פתרון מלא:

הפונקציה: \(f(x) = \frac{3x}{x^2-4}\).

1. תחום הגדרה: המכנה שונה מאפס. \(x^2-4 \neq 0 \Rightarrow x^2 \neq 4 \Rightarrow x \neq \pm 2\).
תחום ההגדרה: \(x \neq 2\) וגם \(x \neq -2\).

2. אסימפטוטות אנכיות: נבדוק את מאפסי המכנה: \(x=2, x=-2\).
המונה \(3x\) אינו מתאפס עבור ערכים אלו (\(3(2)=6 \neq 0\), \(3(-2)=-6 \neq 0\)).
לכן, האסימפטוטות האנכיות הן \(x=2\) ו-\(x=-2\).

3. אסימפטוטה אופקית: החזקה הגבוהה במונה היא 1 (\(3x^1\)). החזקה הגבוהה במכנה היא 2 (\(x^2\)).
מכיוון שחזקת המונה (1) קטנה מחזקת המכנה (2), האסימפטוטה האופקית היא \(y=0\).

תשובה: תחום הגדרה: \(x \neq 2, x \neq -2\). אסימפטוטות אנכיות: \(x=2, x=-2\). אסימפטוטה אופקית: \(y=0\).
