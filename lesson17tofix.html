חקירת פונקציה ריבועית (פרבולה)

לומדים 📚

פונקציה ריבועית היא פונקציה מהצורה \(y = ax^2 + bx + c\), כאשר \(a, b, c\) הם מספרים ממשיים ו-\(a \neq 0\). הגרף של פונקציה ריבועית נקרא פרבולה.

חקירת פונקציה ריבועית כוללת מציאת מאפיינים מרכזיים של הפרבולה, כגון נקודת הקודקוד, ציר הסימטריה, נקודות חיתוך עם הצירים, ותחומי עלייה וירידה.

מיקוד לשנת תשפ"ה:

לפי המיקוד לשאלון 35381, הנושא של "שתי פרבולות" (השוואה ופתרון מערכות של שתי פונקציות ריבועיות) הוסר. נתמקד בחקירה של פרבולה בודדת.

1. כיוון הפרבולה (קעירות)

  • אם \(a > 0\), הפרבולה "צוחקת" (קעורה כלפי מעלה, \(\cup\)), ויש לה נקודת מינימום.
  • אם \(a < 0\), הפרבולה "בוכה" (קעורה כלפי מטה, \(\cap\)), ויש לה נקודת מקסימום.

2. נקודת הקודקוד של הפרבולה

נקודת הקודקוד היא הנקודה הנמוכה ביותר בפרבולה "צוחקת" (מינימום) או הגבוהה ביותר בפרבולה "בוכה" (מקסימום).

שיעור ה-X של הקודקוד, \(x_k\), נתון על ידי הנוסחה:

\(x_k = \frac{-b}{2a}\)

כדי למצוא את שיעור ה-Y של הקודקוד, \(y_k\), מציבים את \(x_k\) שמצאנו במשוואת הפונקציה: \(y_k = a(x_k)^2 + bx_k + c\).

3. ציר הסימטריה

לכל פרבולה יש ציר סימטריה אנכי העובר דרך נקודת הקודקוד. משוואת ציר הסימטריה היא:

\(x = x_k = \frac{-b}{2a}\)

4. נקודות חיתוך עם הצירים

  • חיתוך עם ציר ה-Y: מתקבל כאשר \(x=0\). מציבים \(x=0\) במשוואת הפונקציה: \(y = c\). נקודת החיתוך היא \((0,c)\).
  • חיתוך עם ציר ה-X (נקודות האפס): מתקבל כאשר \(y=0\). פותרים \(ax^2 + bx + c = 0\).

5. תחומי עלייה וירידה, חיוביות ושליליות

  • תחומי עלייה וירידה: נקבעים ביחס לקודקוד \(x_k\).
  • תחומי חיוביות ושליליות: נקבעים ביחס לנקודות החיתוך עם ציר ה-X.

דוגמה מפורטת לחקירת פרבולה (מתוך חומר הבחינה):

נתונה הפונקציה \(y = x^2 - 4x + 3\).

חקירה: (הסברים מפורטים כפי שהיו)

  1. כיוון: \(a=1 > 0\), לכן הפרבולה "צוחקת" (מינימום).
  2. קודקוד: \(x_k = \frac{-(-4)}{2 \cdot 1} = 2\). \(y_k = (2)^2 - 4(2) + 3 = -1\). קודקוד: \((2, -1)\).
  3. ציר סימטריה: \(x = 2\).
  4. חיתוך עם ציר Y: \((0,3)\).
  5. חיתוך עם ציר X: \((1,0), (3,0)\).
  6. תחומי עלייה וירידה: ירידה: \(x < 2\). עלייה: \(x > 2\).
  7. תחומי חיוביות ושליליות: חיובית: \(x < 1\) או \(x > 3\). שלילית: \(1 < x < 3\).

מתרגלים ✍️

תרגיל 1: נתונה הפונקציה \(y = -x^2 + 6x - 5\). מצא את שיעורי נקודת הקודקוד שלה.

פתרון מלא:

\(a=-1, b=6, c=-5\).

\(x_k = \frac{-6}{2 \cdot (-1)} = \frac{-6}{-2} = 3\).

\(y_k = -(3)^2 + 6(3) - 5 = -9 + 18 - 5 = 4\).

נקודת הקודקוד היא \((3,4)\).
