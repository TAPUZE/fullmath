פונקציית שורש (חדו"א) √

לומדים: חקירת פונקציית שורש ריבועי

פונקציית שורש ריבועי היא פונקציה הכוללת ביטוי עם שורש ריבועי, למשל \(f(x) = \sqrt{ax+b}\) או \(f(x) = \sqrt{ax^2+bx+c}\), ולעיתים גם ביטויים נוספים מחוץ לשורש.

1. תחום הגדרה

הכלל החשוב ביותר בפונקציות שורש הוא שהביטוי מתחת לשורש הריבועי חייב להיות אי-שלילי (גדול מאפס או שווה לאפס).

\text{If } f(x) = \sqrt{g(x)}, \text{ then domain is } g(x) \ge 0

לכן, השלב הראשון בחקירת פונקציית שורש הוא פתרון האי-שוויון \(g(x) \ge 0\).

דוגמה (ממסמך המחקר): מצא את תחום ההגדרה של הפונקציה \(f(x) = \sqrt{x-2}\).

פתרון:

נדרוש שהביטוי מתחת לשורש יהיה אי-שלילי:

x-2 \ge 0
x \ge 2

תחום ההגדרה הוא \(x \ge 2\).

2. נגזרת של פונקציית שורש

הנוסחה לגזירת פונקציית שורש ריבועי \(f(x) = \sqrt{u(x)}\) (כאשר \(u(x)\) היא פונקציה גזירה) היא:

f'(x) = (\sqrt{u(x)})' = \frac{u'(x)}{2\sqrt{u(x)}}

חשוב לשים לב שתחום ההגדרה של הנגזרת עשוי להיות קטן יותר מתחום ההגדרה של הפונקציה המקורית, מכיוון שהמכנה \(2\sqrt{u(x)}\) אינו יכול להתאפס (כלומר, \(u(x) > 0\)).

דוגמה (ממסמך המחקר): גזור את הפונקציה \(f(x) = \sqrt{2x+3}\).

פתרון:

נסמן \(u(x) = 2x+3\). לכן, \(u'(x) = 2\).

נשתמש בנוסחת הנגזרת:

f'(x) = \frac{u'(x)}{2\sqrt{u(x)}} = \frac{2}{2\sqrt{2x+3}} = \frac{1}{\sqrt{2x+3}}

תחום ההגדרה של הפונקציה הוא \(2x+3 \ge 0 \Rightarrow x \ge -1.5\).

תחום ההגדרה של הנגזרת הוא \(2x+3 > 0 \Rightarrow x > -1.5\).

3. חקירת פונקציית שורש

החקירה כוללת:

  • תחום הגדרה.
  • נקודות חיתוך עם הצירים (תוך התחשבות בתחום ההגדרה).
  • נקודות קיצון:
    • מצא נקודות חשודות לקיצון פנימיות על ידי פתרון \(f'(x)=0\).
    • בדוק את נקודות הקצה של תחום ההגדרה. אם תחום ההגדרה הוא קטע סגור (למשל \(x \ge a\) או \(a \le x \le b\)), נקודות הקצה יכולות להיות נקודות קיצון (קצה).
    • סווג את הנקודות (מינימום/מקסימום) באמצעות טבלת ערכים או נגזרת שנייה (אם נוח לחשב).
  • תחומי עלייה וירידה.

דוגמה (ממסמך המחקר): חקור את הפונקציה \(f(x) = x - \sqrt{x}\) ומצא נקודות קיצון ותחומי עלייה וירידה.

פתרון:

1. תחום הגדרה: \(x \ge 0\).

2. נגזרת: \(f'(x) = (x)' - (\sqrt{x})' = 1 - \frac{1}{2\sqrt{x}}\). (הנגזרת מוגדרת עבור \(x > 0\)).

3. נקודות חשודות לקיצון:

נשווה נגזרת ל-0 (עבור \(x>0\)): \(1 - \frac{1}{2\sqrt{x}} = 0 \Rightarrow 1 = \frac{1}{2\sqrt{x}} \Rightarrow 2\sqrt{x} = 1 \Rightarrow \sqrt{x} = \frac{1}{2} \Rightarrow x = \frac{1}{4}\).

נקודת הקצה של תחום ההגדרה היא \(x=0\). נבדוק אותה.

4. ערכי הפונקציה בנקודות החשודות:

  • \(f(\frac{1}{4}) = \frac{1}{4} - \sqrt{\frac{1}{4}} = \frac{1}{4} - \frac{1}{2} = -\frac{1}{4}\). נקודה: \((\frac{1}{4}, -\frac{1}{4})\).
  • \(f(0) = 0 - \sqrt{0} = 0\). נקודה: \((0,0)\) (נקודת קצה).

5. תחומי עלייה וירידה (טבלת ערכים עבור \(f'(x) = 1 - \frac{1}{2\sqrt{x}}\)):

נבדוק סימן הנגזרת בתחומים שנוצרים ע"י \(x=\frac{1}{4}\) וקצה התחום \(x=0\):

  • בתחום \(0 < x < \frac{1}{4}\) (למשל, \(x=0.1\)): \(f'(0.1) = 1 - \frac{1}{2\sqrt{0.1}} \approx 1 - \frac{1}{0.632} \approx 1 - 1.58 < 0\) (ירידה).
  • בתחום \(x > \frac{1}{4}\) (למשל, \(x=1\)): \(f'(1) = 1 - \frac{1}{2\sqrt{1}} = 1 - \frac{1}{2} = \frac{1}{2} > 0\) (עלייה).

מכאן, \((\frac{1}{4}, -\frac{1}{4})\) היא נקודת מינימום.

הנקודה \((0,0)\) היא נקודת קצה. מכיוון שהפונקציה יורדת ימינה ממנה, היא מהווה נקודת מקסימום קצה.

תשובה: מינימום ב-\((\frac{1}{4}, -\frac{1}{4})\). מקסימום קצה ב-\((0,0)\).
תחום עלייה: \(x > \frac{1}{4}\).
תחום ירידה: \(0 < x < \frac{1}{4}\).

מתרגלים ✍️

תרגיל 1: מצא את תחום ההגדרה של הפונקציה \(f(x) = \sqrt{9-x^2}\).

פתרון מלא:

נדרוש שהביטוי מתחת לשורש יהיה אי-שלילי:

9-x^2 \ge 0

זוהי פרבולה "בוכה" (מקדם \(x^2\) שלילי). נמצא את שורשיה:

9-x^2 = 0 \Rightarrow x^2 = 9 \Rightarrow x = \pm 3

הפרבולה חיובית או שווה לאפס בין השורשים (כולל).

תשובה: תחום ההגדרה הוא \(-3 \le x \le 3\).
