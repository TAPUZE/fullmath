חיתוך של מעגל וישר ✂️

לומדים: מציאת נקודות חיתוך בין מעגל לישר

נקודות החיתוך בין מעגל לישר הן הנקודות המשותפות לשתי הצורות, כלומר, הנקודות שמקיימות גם את משוואת המעגל וגם את משוואת הישר. כדי למצוא אותן, עלינו לפתור מערכת של שתי משוואות:

  • משוואת המעגל (משוואה ריבועית): \((x-a)^2 + (y-b)^2 = R^2\) או \(x^2 + y^2 = R^2\).
  • משוואת הישר (משוואה לינארית): \(y = mx + n\) או \(Ax + By + C = 0\).

1. שיטת הפתרון: הצבה

הדרך הנפוצה לפתור מערכת כזו היא באמצעות הצבה:

  1. בטא משתנה אחד: בטא את אחד המשתנים (בדרך כלל \(y\)) באמצעות המשתנה השני מתוך משוואת הישר. לדוגמה, אם משוואת הישר היא \(y = mx + n\), הביטוי עבור \(y\) מוכן.
  2. הצב במשוואת המעגל: הצב את הביטוי שמצאת בסעיף הקודם במקום המתאים במשוואת המעגל.
  3. פתור את המשוואה הריבועית: לאחר ההצבה, תקבל משוואה ריבועית בנעלם אחד (בדרך כלל \(x\)). פתור אותה כדי למצוא את שיעורי ה-\(x\) של נקודות החיתוך.
  4. מצא את שיעורי ה-\(y\): הצב כל אחד מערכי ה-\(x\) שמצאת בחזרה במשוואת הישר (הפשוטה יותר) כדי למצוא את שיעורי ה-\(y\) המתאימים.
  5. רשום את נקודות החיתוך.

2. מספר הפתרונות האפשרי (מצבים הדדיים)

לאחר ההצבה וקבלת משוואה ריבועית מהצורה \(Ax^2 + Bx + C = 0\), מספר הפתרונות (ולכן מספר נקודות החיתוך) תלוי בדיסקרימיננטה \(\Delta = B^2 - 4AC\):

  • אם \(\Delta > 0\): שני פתרונות שונים ל-\(x\), ולכן שתי נקודות חיתוך (הישר חותך את המעגל).
  • אם \(\Delta = 0\): פתרון יחיד ל-\(x\) (שורש כפול), ולכן נקודת חיתוך אחת (הישר משיק למעגל).
  • אם \(\Delta < 0\): אין פתרונות ממשיים ל-\(x\), ולכן אין נקודות חיתוך (הישר אינו פוגש את המעגל).

2 נקודות חיתוך (\(\Delta > 0\))

נקודת חיתוך אחת (משיק, \(\Delta = 0\))

אין נקודות חיתוך (\(\Delta < 0\))

דוגמה פתורה (ממסמך המחקר)

דוגמה: מצא את נקודות החיתוך של המעגל \(x^2 + y^2 = 25\) והישר \(y = x + 1\).

פתרון:

1. נציב את משוואת הישר (\(y = x+1\)) במשוואת המעגל (\(x^2 + y^2 = 25\)):

x^2 + (x+1)^2 = 25

2. נפתח סוגריים ונסדר את המשוואה:

x^2 + (x^2 + 2x + 1) = 25
2x^2 + 2x + 1 - 25 = 0
2x^2 + 2x - 24 = 0

נחלק את כל המשוואה ב-2 כדי לפשט:

x^2 + x - 12 = 0

3. נפתור את המשוואה הריבועית: ניתן להשתמש בפירוק לגורמים או בנוסחת השורשים.

פירוק לגורמים: \((x+4)(x-3) = 0\).

מכאן, הפתרונות ל-\(x\) הם:

x_1 = -4 \quad \text{or} \quad x_2 = 3

4. נמצא את שיעורי ה-\(y\) המתאימים על ידי הצבה במשוואת הישר \(y = x+1\):

עבור \(x_1 = -4\): \(y_1 = -4 + 1 = -3\).

עבור \(x_2 = 3\): \(y_2 = 3 + 1 = 4\).

5. נקודות החיתוך הן \((-4, -3)\) ו-\((3, 4)\).

תשובה: נקודות החיתוך הן \((-4, -3)\) ו-\((3, 4)\).

מתרגלים ✍️

תרגיל 1: מצא את נקודות החיתוך של המעגל \((x-2)^2 + y^2 = 5\) והישר \(y = x-1\).

פתרון מלא:

1. נציב \(y = x-1\) במשוואת המעגל:

(x-2)^2 + (x-1)^2 = 5

2. נפתח ונסדר:

(x^2 - 4x + 4) + (x^2 - 2x + 1) = 5
2x^2 - 6x + 5 = 5
2x^2 - 6x = 0

3. נוציא גורם משותף:

2x(x - 3) = 0

מכאן, \(x_1 = 0\) או \(x_2 = 3\).

4. נמצא \(y\) עבור כל \(x\) באמצעות \(y = x-1\):

עבור \(x_1 = 0 \Rightarrow y_1 = 0-1 = -1\). נקודה: \((0, -1)\).

עבור \(x_2 = 3 \Rightarrow y_2 = 3-1 = 2\). נקודה: \((3, 2)\).

תשובה: נקודות החיתוך הן \((0, -1)\) ו-\((3, 2)\). (סדר הנקודות אינו משנה בכתיבת התשובה).

בחן את עצמך 🧐
